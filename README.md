# 我的专属收藏夹 · 智能折叠拖拽版

一款美观、实用、交互友好的书签管理工具，支持分类展示、拖拽排序、访问频率统计、自定义编辑等功能，融合国风配色与现代UI设计，适配移动端与桌面端。

## ✨ 核心功能

### 1. 基础功能
- **分类分组展示**：按自定义分类展示书签，每个分类拥有独特的国风配色
- **多维度排序**：支持按访问频率（升/降序）、名称（A-Z/Z-A）排序
- **分类筛选**：快速筛选查看指定分类的书签
- **访问频率统计**：自动记录每个书签的访问次数，可一键重置

### 2. 高级交互
- **智能折叠拖拽**：拖拽分类时自动折叠其他分类内容，聚焦拖拽操作，提升排序体验
- **自定义编辑模式**：支持添加、修改、删除书签，编辑分类名称
- **拖拽排序记忆**：分类排序结果自动保存到本地存储，刷新不丢失

### 3. 特色功能
- **息屏保护**：闲置30秒自动触发息屏模式，显示精美诗词与实时时间
- **响应式设计**：完美适配移动端与桌面端设备
- **本地存储**：所有书签数据保存在浏览器本地，无需服务器，隐私安全

## 🎨 设计亮点
- **国风配色系统**：每个分类配备专属的国风配色方案，视觉层次分明
- **精美动效**：书签卡片悬浮动效、拖拽幽灵元素、平滑过渡动画
- **人性化交互**：拖拽时的视觉反馈、编辑模式的清晰区分、操作确认提示

## 🚀 快速使用

### 本地部署
1. 复制全部代码保存为 `index.html` 文件
2. 直接用浏览器打开该文件即可使用
3. 所有数据自动保存在浏览器本地存储中

### 基础操作
#### 浏览书签
- 点击书签链接可直接打开（自动统计访问次数）
- 使用顶部筛选器选择分类，排序下拉框切换排序方式

#### 拖拽排序
- 直接拖动分类标题栏即可调整分类顺序
- 拖拽时其他分类自动折叠，聚焦当前操作

#### 编辑书签
1. 点击「开启自定义编辑模式」按钮
2. 可修改书签名称、网址、分类，或删除书签
3. 填写表单可添加新书签（需包含完整网址）
4. 编辑完成后点击「保存编辑内容」生效

#### 息屏功能
- 页面闲置30秒自动进入息屏模式
- 点击息屏界面任意位置退出息屏模式

## 📱 适配说明
- 桌面端：完整展示所有功能，网格布局展示书签
- 移动端：自适应布局，分类标题纵向排列，筛选/排序控件堆叠展示

## 🎯 技术特性
- 原生HTML/CSS/JavaScript实现，无第三方依赖
- 使用CSS变量实现动态配色系统
- 原生拖拽API结合自定义幽灵元素，提升拖拽体验
- 本地存储（localStorage）持久化数据
- 事件委托与防抖处理，优化性能

## 📋 自定义扩展

### 添加新分类配色
在CSS中添加新的分类配色规则：
```css
.category-新分类名称 { 
    --category-color: #颜色值; /* 主色 */
    --category-bg: #颜色值;    /* 背景色 */
}
```

### 修改息屏诗词
编辑JavaScript中的`philosophyQuotes`数组，添加自定义诗词：
```javascript
const philosophyQuotes = [
    "你的自定义诗词1 —— 作者",
    "你的自定义诗词2 —— 作者"
];
```

### 修改闲置触发时间
修改`IDLE_TIME`常量（单位：毫秒）：
```javascript
const IDLE_TIME = 30000; // 30秒，可自定义
```

## 💡 注意事项
1. 书签数据存储在浏览器本地，清除浏览器数据会导致书签丢失
2. 添加书签时需填写完整网址（包含http:///https://）
3. 拖拽排序仅在非编辑模式下可用
4. 编辑模式下修改的内容需点击保存才会永久生效
